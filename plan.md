# 공지사항 첨부파일 및 이미지 지원 구현 계획

## 1. 완료한 작업
- 관리자 공지 작성 화면에 이미지 업로드 버튼 추가
- 관리자 공지 작성 화면에 PDF 첨부 버튼 추가
- Firebase Storage 연동
- 공지 상세 화면에서 이미지와 PDF 첨부 링크 표시
- 공지 데이터 모델에 attachments 필드 추가

## 2. 남은 운영 작업
1. Firebase Storage 규칙 배포
2. 실제 Firebase 프로젝트에 Storage 활성화
3. 관리자 계정으로 업로드 테스트
4. 필요 시 이미지 최적화 또는 썸네일 생성 추가

## 3. 배포 순서
- 1단계: Firebase 콘솔에서 Storage 활성화
- 2단계: storage.rules 배포
- 3단계: 앱 빌드 및 배포
- 4단계: 관리자 페이지에서 이미지/PDF 업로드 확인

## 4. 향후 개선 아이디어
- 본문 이미지에 설명 텍스트와 정렬 옵션 추가
- 첨부파일 목록에 삭제/다운로드 상태 표시
- Rich Text Editor 도입으로 이미지 드래그 앤 드롭 지원
